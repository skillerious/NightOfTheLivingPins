Use base materials for structure and decals for storytelling. Decals let you add grime, cracks, stains, and damage without building everything into the mesh.

Texture import workflow

  1. Create Content/NightOfTheLivingPins/Textures and subfolders for Lane, Walls, Metal, Decals, and Props.
  2. Import one complete texture set at a time: base color, normal, roughness, masks, emissive, and opacity if needed.
  3. Open each imported texture and check the settings before using it in a material.
  4. Keep sRGB on for base color and emissive color textures.
  5. Keep sRGB off for roughness, metallic, ambient occlusion, masks, and packed utility textures.
  6. Apply the texture set to a material instance, then test it on a blockout cube before spreading it across the level.

Use BC for base color, N for normal, R for roughness, AO for ambient occlusion, and Masks for packed grayscale channels.

Where to use decals

  • Shoe scuffs near the approach.
  • Oil streaks and wear down the lane edges.
  • Water leaks on side walls and ceiling.
  • Blood drag marks near the back machine.
  • Handprints on ball return and scoring monitor.

Where not to use decals

  • Do not cover the whole lane in blood.
  • Do not place high-contrast stains over aiming arrows.
  • Do not make all surfaces equally noisy.
  • Leave visual rest areas so the important props stand out.

Material editor workflow

  1. Open the Content Drawer and go to Materials.
  2. Right-click and create a Material, then name it with the M_ prefix.
  3. Double-click it to open the Material Editor.
  4. Create Material Instances in Materials/Instances so you can tune color, roughness, wetness, and grime without rebuilding the base material.
  5. Select a mesh actor and assign the instance in Details > Materials.

Decal placement workflow

  1. Open Place Actors > Visual Effects and drag a Decal into the level.
  2. Select it, then use Details to assign the decal material.
  3. Use Location, Rotation, and Scale to project it onto the wall, floor, lane edge, or back machine.
  4. Move decal actors into 04_PROPS_DRESSING or a dedicated decal folder so they stay manageable.

Base material: rotten wood, concrete, metal, ceramic, rubber, and old plastic. Keep broad values simple.
Surface variation: roughness masks, wet strips, cracked boards, chipped paint, rust gradients.
Story decals: slime drips, handprints, shoe scuffs, oil streaks, wall leaks, pin deck stains.
Impact decals: blood, splinters, scrape marks, burst stains, and danger signage near the back machine.
Zombie pin material notesFrom supplied pin art
  • Ceramic shell: off-white, cracked, chipped, dirty, and slightly glossy at worn edges.
  • Red band: faded paint with breaks and blood crossing the stripe.
  • Face cut-in: skull planes and glowing eyes need strong contrast against the pin body.
  • Slime: green/yellow translucent drips should catch light but not cover the silhouette.
