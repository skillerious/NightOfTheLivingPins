Get comfortable with the Unreal Editor interface first

Before building Lane 13, learn the few panels you will touch constantly. Most level work follows the same loop: place an Actor, select it, rename it, set its transform in Details, organize it in the Outliner, then save the level.

Unreal Editor working layoutThe panels used by this guide

If your editor does not look like this, use Window > Load Layout > Default Editor Layout, then reopen any missing panel from the Window menu.

Menu BarTop-left menus such as File, Edit, Window, Tools, and Help. Use this for new levels, project settings, opening panels, and restoring layouts.
Main ToolbarTop toolbar for Save, mode selection, content shortcuts, Blueprints, Cinematics, and Play In Editor.
OutlinerUpper-right list of everything in the current level. Use it to find, select, rename, hide, and folder your level actors.
Level ViewportThe large 3D view where you place and move actors. Left-click selects. Hold Right Mouse Button and use W/A/S/D to fly around. Press F to frame the selected actor.
Details PanelRight-side properties for the selected actor. This is where you type exact Location, Rotation, Scale, Material, Physics, Collision, Light, Tags, and Mobility values.
Content Drawer / Content BrowserBottom-left asset browser. Press Ctrl + Space to open it, then dock it if you want it to stay open while you make folders and assets.

The actor workflow

  1. Open Place Actors from Window > Place Actors.
  2. Drag the actor type into the Level Viewport.
  3. Select it in the viewport or Outliner.
  4. Rename it at the top of the Details panel or in the Outliner.
  5. Type exact Location, Rotation, and Scale values in Details.
  6. Move it into the correct Outliner folder.
  7. Save the level with Ctrl + S.

Recover missing panels

  • Outliner: Window > Outliner.
  • Details: Window > Details.
  • Place Actors: Window > Place Actors.
  • Content Drawer: bottom-left button or Ctrl + Space.
  • Default layout: Window > Load Layout > Default Editor Layout.

Viewport movement basics

  • Hold Right Mouse Button and use WASD to fly.
  • Use Q and E while holding Right Mouse Button to move down and up.
  • Mouse wheel changes distance; Right Mouse Button plus wheel changes camera speed.
  • Select an actor and press F to focus the camera on it.

Transform tool basics

  • Q selects without a gizmo.
  • W moves the selected actor.
  • E rotates the selected actor.
  • R scales the selected actor.
  • Space cycles move, rotate, and scale.
  • Use Details for exact values; use the viewport gizmo for rough positioning.

View and test basics

  • Use the viewport's Perspective menu for top/front/side orthographic views.
  • Use Lit, Unlit, and Wireframe to diagnose lighting or blockout shape.
  • Press G for Game View when you want to hide editor icons.
  • Use the Main Toolbar Play button to test inside the editor.
When this guide says...Do this in the editorCheck before moving on
Create a Content folderOpen Content Drawer, right-click inside Content or your project folder, choose New Folder, then name it exactly.The folder appears under Content/NightOfTheLivingPins, not loose in a random plugin or engine folder.
Create an Outliner folderIn the Outliner, use the folder button or right-click empty space and create a folder. Drag selected actors into it.Actors are grouped by purpose: blockout, lighting, props, gameplay markers, triggers, FX, audio, cameras.
Place a cubeOpen Place Actors, choose Shapes, drag Cube into the viewport, then set exact transform values in Details.The actor name, Location, and Scale match the table before you duplicate it.
Add a Target PointOpen Place Actors, choose Basic, drag Target Point into the viewport, rename it, then type its Location in Details.Target Points sit where future code expects ball, pin, camera, enemy, and scare positions.
Add a Trigger VolumeOpen Place Actors, choose Volumes, drag a Trigger Volume or Box Trigger, then resize it in Details.It covers the gameplay area cleanly without relying on messy art mesh collision.
Assign a materialSelect the mesh actor, open Details, find the Materials section, and choose the material or material instance slot.The lane center remains readable after applying grime, roughness, wetness, or decals.
Important mental model: an Actor is something placed in the level. The Outliner tells you what actors exist. The Details panel tells you what the selected actor is doing. The Content Drawer stores reusable assets that can be placed as actors.
Beginner safety rule: do not eyeball transforms for the core lane. Use the viewport to get close, then type exact numbers into Details so the bowling scale, pin rack, gutter triggers, and camera markers stay reliable.
