Daily editor warm-up

  1. Open the template map.
  2. Open or dock the Content Drawer.
  3. Check Outliner folders are visible.
  4. Select one actor and confirm Details appears.
  5. Press Play once before making big changes.

Common shortcuts

  • Ctrl + S: save current level.
  • Ctrl + Space: Content Drawer.
  • W: move.
  • E: rotate.
  • R: scale.
  • F: focus selected actor.
  • G: Game View.

Panel troubleshooting

  • If Details is blank, select an actor first.
  • If the Content Drawer keeps closing, dock it.
  • If a volume is hard to see, select it in Outliner.
  • If the viewport is slow, lower viewport scalability while editing.
  • If the layout feels broken, reload the default layout.
Final milestone: before moving into proper gameplay programming, you want a greybox haunted bowling lane where a temporary physics ball can roll into temporary physics pins. Once that is satisfying, the art pass becomes safer because the level already works.
