Install Unreal Engine and create the project

Use Epic Games Launcher, create the UE5 project, choose a clean folder location, save the first map, and build the project folders before importing anything.

Learn the Unreal Editor panels

Get comfortable with the Viewport, Outliner, Details panel, Place Actors, Content Drawer, transforms, save habits, and Play In Editor testing.

Editor 02

Lock the lane scale before art

Use the real bowling lane dimensions as the core gameplay spine, then make the surrounding room wider and more theatrical for horror composition.

Scale 03

Block out the playable space with cubes

Place the approach, lane, gutters, rails, pin deck, walls, ceiling, and back machine using exact transforms so gameplay has a reliable base.

Greybox 04

Keep the ball path visible, push horror contrast to the back machine and side walls, then use fog and post process only after the lighting pass is legible.

How to use this guide if you are new to Unreal

Do not try to make the final horror scene in one pass. Work through the guide in order and only move on when the current pass has a clear, testable result.

Beginner workflow

Read one page, then build it

Keep the relevant page open beside Unreal. Build the exact step, save the map, then continue. Avoid jumping from lighting to props before the blockout works.

Type exact values first

When a table gives transforms, type those numbers into Details. Eyeballing is fine for props, but the lane, pins, triggers, and cameras need repeatable values.

Use temporary assets proudly

A sphere for the ball and simple placeholder pins are correct at this stage. Replace them only after throw, collision, camera, and result flow are readable.

Check the Outliner daily

If the Outliner fills with Cube, Box, Boolean, and Sphere names, pause and rename. Good organization is not polish; it is how you keep the project understandable.

Test in Play In Editor

After every major pass, press Play and test the level from the player's point of view. The lane should feel playable before the scene feels beautiful.

Stop each pass on purpose

Do not endlessly decorate. Finish setup, then scale, then blockout, then markers, then lighting, then textures, then dressing, then reuse.

PR

Professional layout rules for the build

These rules keep the level and this guide readable. They are repeated across the pages because the project needs detail, but not visual overload.

Readable art pass

One primary path

The center of the lane must stay readable from approach to pins. Put heavy horror detail beside, above, and behind the lane.

Measured first

Do not texture or dress a broken layout. Exact transforms, target points, and collision volumes come before beauty work.

Texture in layers

Start with broad materials, then add texture variation, then decals, then focal-point damage. Avoid making every surface equally noisy.

Reusable by default

Name actors clearly, keep Outliner folders tidy, and preserve a clean template map so new lane variants do not inherit messy one-off decisions.
